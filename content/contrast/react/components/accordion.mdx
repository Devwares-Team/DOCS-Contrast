# React Bootstrap 5 Accordion

React Bootstrap 5 Accordion is a UI element that consists of vertically stacked lists that hides and exposes text blocks, pieces of information with a single click. This piece of design element makes for a great user experience as they increase interaction between the user and the website.

As a result of this component toggling the React Bootstrap 5 Accordion are useful in websites as they reduce the number of page a user has to scroll through.

**Applications of Accordions on a website**

- Accordions can be used in the frequently asked questions (FAQ) section of the website.
- Accordions can be used in pages showcasing the products offered on an online stores.
- Data Tables can be constructed using the React Bootstrap 5 accordion component.

## Importing the React Bootstrap 5 Accordion component

To use the React Bootstrap 5 Accordion component, you would need to import it into your project.

```js
import { CDBAccordion } from 'cdbreact';
```

- `title` : This prop takes in a parameter that defines the title of the accordion section.

# Contrast React Bootstrap 5 Accordion Props

This section will build on your information about the props you get to use with the CDBReact Bootstrap 5 Accordion component. You will find out what these props do, their default values, and how you would use them in your code.

The table below lists the prop options of the `CDBAccordion` component to personalize your Accordion.

| Name                 |  Type   | Default |                 Description                  |                                                   Example |
| :------------------- | :-----: | ------: | :------------------------------------------: | --------------------------------------------------------: |
| accordionClass       | String  |         |             Adds custom classes              |             <CDBAccordion accordionClass="myClass" ... /> |
| accordionHeaderClass | String  |         | Adds custom classes for the accordion header | <CDBAccordion accordionHeaderClass="myHeaderClass" ... /> |
| accordionBodyClass   | String  |         |  Adds custom classes for the accordion body  |     <CDBAccordion accordionBodyClass="myBodyClass" ... /> |
| tag                  | String  |     div |        Changes default Accordion tag         |                      <CDBAccordion tag="Accordion" ... /> |
| hideIcon             | Boolean |   false |             Hides Accordion icon             |                              <CDBAccordion hideIcon .../> |
