# React Bootstrap ListGroup

React Bootstrap List group is a versatile component that can display both simple and complex custom information.

## Importing the React Bootstrap ListGroup component

Start by importing `CDBListGroup` into your project to use the Contrast React Bootstrap ListGroup component. Then you import the `CDBListGroupItem` for the list group's items.

```javascript
import { CDBListGroup, CDBListGroupItem } from "cdbreact";
```

## List Group as Link

The `href` attribute on the `CDBLIstGroupItem` makes the list group item behave as a `Link` element.

<div class="container">
  <ul class="list-group" style="width: 28rem">
    <li class="list-group-item list-group-item-action active">List Group</li>
    <li class="list-group-item">List Group</li>
    <li class="list-group-item">List Group</li>
    <li class="list-group-item">List Group</li>
    <li class="list-group-item">List Group Link</li>
    <li class="list-group-item">List Group</li>
  </ul>
</div>


```javascript
import React from "react";
import { CDBListGroup, CDBListGroupItem, CDBContainer } from "cdbreact";

const LIstGroup = () => {
  return (
    <CDBContainer>
            <CDBListGroup style={{ width: "30rem" }}>
              <CDBListGroupItem href="#" active>
                List Group
              </CDBListGroupItem>
              <CDBListGroupItem href="#" hover>
                List Group
              </CDBListGroupItem>
              <CDBListGroupItem href="#" hover>
                List Group
              </CDBListGroupItem>
              <CDBListGroupItem href="#" hover>
                List Group
              </CDBListGroupItem>
              <CDBListGroupItem href="#" hover>
                List Group Link
              </CDBListGroupItem>
              <CDBListGroupItem href="#" hover>
                List Group
              </CDBListGroupItem>
            </CDBListGroup>
    </CDBContainer>
  );
};
export default  LIstGroup;
```

# API

This section will explain the props available with the Contrast React Bootstrap ListGroup component. You will discover what these props do, their default values, and how you would use them in your code.

## API Reference: Contrast React Bootstrap ListGroup Props

Other CDBListGroup component prop options are listed in the table below.

| Name            | Type        | Default      |   Description| Example      |
| :------------- | :----------: | -----------: | :----------: | -----------: |
| className      | String       |              |Adds custom classes	      |     <CDBLIstGroup className="myClass" ... />  |
| tag            | function or String       | ul          | Changes default input tag | <CDBLIstGroup tag="ul" ... />  |

## API Reference: Contrast React Bootstrap LIstGroupItem Properties

Other prop options for the `CDBListGroupItem` component are listed in the table below.

| Name            | Type        | Default      |   Description| Example      |
| :------------- | :----------: | -----------: | :----------: | -----------: |
| className      | String       |              |Adds custom classes	      |     <CDBLIstGroupItem className="myClass" ... />  |
| tag            | function or String       | li          | Changes default input tag | <CDBLIstGroupItem tag="li" ... />  |
| active        | Boolean      | false        | Adds default active state to item | <CDBLIstGroupItem active .../>  |
| color            | String       |           | Changes Item background and text color, accepts `["primary", "default", "secondary", "success", "dark", "danger", "info", "warning", "white" ]` | <CDBLIstGroupItem color="primary" ... />  |
| disable        | Boolean      | false        | Disables item from being clicked | <CDBLIstGroupItem disabled .../>  |
| hover        | Boolean      | false        | Adds hover effect (slightly background color change) | <CDBLIstGroupItem hover .../>  |
